# web-client

`traffic-control-system`용 React(Vite) 프론트엔드 프로젝트입니다.

## 실행

```bash
cd web-client
npm install
npm run dev
```

기본 개발 서버는 `http://localhost:5173` 입니다.

## 기능

- API Gateway URL 입력
- `GET /api/v1/concerts/seats` 호출
- 좌석 목록 렌더링

## 참고

- 이 프로젝트는 API 호출 대상을 `api-gateway`로 가정합니다.
- 예시 기본값: `http://localhost:8080`
